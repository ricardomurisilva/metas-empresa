# Metas Empresa

Sistema simples para controle e acompanhamento de metas empresariais.

## Objetivo
Organizar e acompanhar metas de forma prática, permitindo análise de desempenho e evolução ao longo do tempo.

## Funcionalidades (inicial)
- Cadastro de metas
- Registro de resultados
- Acompanhamento de progresso
- Estrutura base para futuras automações

## Estrutura inicial
- src/ → lógica do sistema
- data/ → armazenamento de dados
- docs/ → documentação complementar

## Próximos passos
- Implementar cálculo de atingimento de metas
- Criar relatórios simples
- Evoluir para interface web

Projeto em desenvolvimento com foco em aprendizado de versionamento (Git), organização de código e práticas de DevOps.
